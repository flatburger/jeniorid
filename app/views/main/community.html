OpenStack

OpenStack Indonesia adalah sebuah komunitas untuk para peminat teknologi cloud computing di Indonesia.OpenStack indonesia ini dibangun dengan tujuan untuk saling berbagi informasi tentang platform cloud khususnya OpenStack.
